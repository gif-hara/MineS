# 修羅の洞窟
## 概要
- 序盤から高レベルの敵が出現する装備品を鍛え上げた状態で挑んで欲しいダンジョン
- ストーリーダンジョンの最後的な立ち位置
- アイテムの持ち込みは可能だけど無しでもクリア出来る構成にしたい
- ご褒美アイテムとして`41F~50F`にショップで以下のアイテムが売られている可能性があります
    - `怪剣シュテンドウジ`
    - `大器の剣`
    - `金剛タマモノマエ`
    - `大器の盾`

## 構成
### 基本情報
- ハートアイテム数
    - `0 ~ 2`
- 金床数
    - `0 ~ 2`
- お金が落ちている数
    - `3 ~ 5`
- お金の容量
    - `10 ~ 500`
- 敵の出現数
    - `8 ~ 10`
- アイテムが落ちている数
    - `2 ~ 3`
- 罠
    - `0 ~ 5`
### 階数
- 50F
