# 腕鳴らしのダンジョン
## 概要
- [初めてのダンジョン](../ElementaryLevel/README.md)をクリアすることで開放されるダンジョンです
- 階数が増えましたがノリは初めてのダンジョンと変わりません
- ここで初めて`罠`が登場します
- `18F`にてショップが確定で登場します
    - ショップに訪れた場合、それ以降村でも登場するようになります

## 構成
### 基本情報
- ハートアイテム数
    - `1 ~ 5`
- 金床数
    - `2 ~ 4`
- お金が落ちている数
    - `3 ~ 5`
- お金の容量
    - `10 ~ 200`
- 敵の出現数
    - `6 ~ 10`
- アイテムが落ちている数
    - `2 ~ 4`
- 罠
    - `0 ~ 2`
### 階数
- 20F
