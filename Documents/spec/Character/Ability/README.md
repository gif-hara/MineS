## 特殊能力について
- 特殊能力とは武器、盾、アクセサリーに付与されているスキルのようなものです
- 敵も特殊能力を持っている場合があります
- 重複して効果が発揮する特殊能力もあります
- 相手に対して影響のある特殊能力は、倒された場合は発動しません
- **※調整する可能性があるため、値に整合性が無い可能性もあります**
    - 最新の状態を見たい場合は関連ソースコードを確認してください:eyes:

## 種類

### 貫通
- アーマーを無視して直接ヒットポイントへダメージを与えます

### 慈悲
- ターン経過時に他の敵のヒットポイントを回復します
- 回復量は`(HPMax / 10)`です
    - :eyes:`Calculator.GetMercyAbilityValue();`

### 自己再生
- ターン経過時にヒットポイントを回復します
- 回復量は`(abilityNumber * 2)`です
    - :eyes:`Calculator.GetRegenerationAbilityValue();`

### 跳返
- 受けたダメージの一部を相手に与えます
- ダメージ量は最小で`受けたダメージ / 20`、最大で`受けたダメージ / 10`になります
    - :eyes:`Calculator.GetSplashDamage();`

### 吸収
- 与えたダメージの一部を回復します
- 回復量はプレイヤーの場合は`与えたダメージ / 3`、敵の場合は`与えたダメージ / 2`になり、回復量が`0`以下の場合は`1`になります
    - :eyes:`Calculator.GetAbsorptionRecovery();`

### 遠距離攻撃
- ターン経過時に攻撃を行います
- `敵にのみ付与される特殊能力`です
- ターン経過時なので敵が発見された時にも攻撃を行います

### 三子教訓
- フィールド上に出現している敵の数だけ攻撃力が上昇します
- 上昇量は`(abilityNumber * (敵の数 - 1)) * 2`です
    - :eyes:`Calculator.GetReinforcementValue();`

### 一致団結
- フィールド上に出現している敵の数だけ受けるダメージが減少します
- さらに他の敵にも同様の効果が適用されます
- `敵のみ付与される特殊能力`です
- 減少量は`((敵の数 - 1) * (全ての敵の一致団結の所持数 * 2))`です
    - :eyes:`Calculator.GetAbilityUnitySubDamage()`

### 連撃
- 攻撃した際に他にも敵がいた場合に与えたダメージの半分を与えます
- このダメージは特殊能力の影響を受けません
    - 例えば`貫通`を持っていてもアーマーに対してダメージを与えます

### 習熟
- 取得する経験値が上昇します
- `プレイヤーにのみ付与される特殊能力`です
- 上昇量は`(経験値 * (abilityNumber / 10))`です
    - :eyes:`Calculator.GetProficiencyRate();`

### 巻上
- 取得するお金が上昇します
- 上昇量は`(お金 * (abilityNumber / 5))`です
    - :eyes:`Calculator.GetHoistRate();`

### 改修
- 敵を倒した際にアーマーが回復します
- 回復量は`((abilityNumber * 5) + 2)`です
    - :eyes:`Calculator.GetRepairValue();`

### 幸運
- 敵を倒した際のアイテムドロップ率が上昇します
- 上昇率は`(abilityNumber)%`となります
    - :eyes:`calculator.GetFortuneItemDropValue();`

### 匠
- 攻撃力が上昇します
- 上昇量は`((abilityNumber * 4) + 2)`です
    - :eyes:`Calculator.GetArtisanRate();`

### 精巧
- アーマーが上昇します
- 上昇量は`((abilityNumber * 8) + 7)`です
    - :eyes:`Calculator.GetExquisiteArmorValue();`

### 回避
- 回避率が上昇します
- 上昇率は`(abilityNumber)%`です
    - :eyes:`Calculator.GetEvationAbilityValue();`

### 五右衛門
- 与えたダメージの一部をお金に加算します
- 取得量は`(Random.Range(10, abilityNumber * 100) + 1)`です
    - :eyes:`Calculator.GetGoemonValue();`

### 窃盗
- 攻撃した際に確率でアイテムを取得します
- `プレイヤーのみ付与される特殊能力`です
- アイテム欄に空きが無い場合は発動しません
- 取得に成功した場合は敵の位置から取得演出が発生します
- 取得するアイテムはダンジョンに出てくるアイテムから選出されます
- 取得確率は`(abilityNumber + 2)`です
    - :eyes:`Calculator.IsSuccessTheft();`

### 必中
- 攻撃が必ず当たるようになります
- ただし状態異常の`恐怖`になっている場合は発動しません
    - :eyes:`CharacterData.CanAttack()`

### 捨身
- アーマーを消費して攻撃力が上昇します
- アーマーが無い場合は発動しません
- 攻撃力上昇量は`(abilityNumber * 10)`です
    - :eyes:`Calculator.GetRiskOfLifeAddStrengthValue();`
- 攻撃する際に消費されるアーマー値は`((abilityNumber / 2) + 1)`です
    - :eyes:`Calculator.GetRiskOfLifeSubArmorValue();`

### 増進
- バフの継続ターンが上昇します
- 継続ターンはバフが付与されたタイミングで決められます
- 上昇量は`(abilityNumber * 2)`です
    - :eyes:`Calculator.GetEnhancementAddTurn();`

### 減退
- バフの継続ターンが減少します
- 継続ターンはバフが付与されたタイミングで決められます
- 減少量は`(abilityNumber * 2)`です
    - :eyes:`Calculator.GetWeakSubTurn();`
- これ使われていないかも

### 免疫
- デバフの継続ターンが減少します
- 継続ターンはデバフが付与されたタイミングで決められます
- 減少量は`(abilityNumber * 2)`です
    - :eyes:`Calculator.GetImmunitySubTurn();`

### 伝染
- デバフの継続ターンが上昇します
- 継続ターンはデバフが付与されたタイミングで決められます
- 上昇量は`(abilityNumber * 2)`です
    - :eyes:`Calculator.GetInfectionAddTurn();`
- これ使われていないかも

### 罠避
- 罠に引っかからなくなります
- `プレイヤーのみ付与される特殊能力`です

### 薬師
- 回復系アイテムの回復量が上昇します
- `プレイヤーのみ付与される特殊能力`です
- 上昇量は`((abilityNumber * 15) + 10)%`です
    - :eyes:`Calculator.GetHealingBuddhaValue();`

### 毒塗
- ダメージを与えた際に確率でデバフの毒を付与します
- 付与する確率はプレイヤーの場合は`((abilituNumber * 2) + 10)%`で、敵の場合は`((abilityNumber * 10) + 10)%`です
    - :eyes:`Calculator.CanAddAbnormalStatusFromAbility();`

### 放心
- ダメージを与えた際に確率でデバフの呆けを付与します
- 付与する確率は`毒塗`と同様です

### 秘孔突き
- ダメージを与えた際に確率でデバフの負傷を付与します
- 付与する確率は`毒塗`と同様です

### 刃毀
- ダメージを与えた際に確率でデバフの鈍らを付与します
- 付与する確率は`毒塗`と同様です

### 錯乱
- ダメージを与えた際に確率でデバフの混乱を付与します
- 付与する確率は`毒塗`と同様です

### 脅迫
- ダメージを与えた際に確率でデバフの恐怖を付与します
- 付与する確率は`毒塗`と同様です

### 呪縛
- ダメージを与えた際に確率でデバフの封印を付与します
- 付与する確率は`毒塗`と同様です

### 雄叫び
- 敵が出現した際に他の敵も出現させます
- `敵のみ付与される特殊能力`です
- 他の敵がいなかったり、抽選結果でロックされたセルに敵がいる場合は発動しません

### 回復
- ダメージを与えた際にヒットポイントが回復します
- 回復量は`(abilityNumber * 3)`です
    - :eyes:`Calculator.GetAbilityRecoveryValue();`

### 防護
- 受けるダメージを減少します
- 減少量は`(abilityNumber * 3)`です
    - :eyes:`Calculator.GetProtectionSubDamage();`

### 先手
- 敵が出現した際にダメージを与えます
- タップで出現した敵にのみ適用されます
    - 例えば`雄叫び`で出現した敵には適用されません
- `プレイヤーのみ付与される特殊能力`です
- ダメージ量は`(abilityNumber * 9)`です
    - :eyes:`Calculator.GetInitiativeDamage();`

### 千里眼
- 常に状態異常の透視の効果を発揮します

### 分裂
- ダメージを受けた際に同じ種類の敵をダンジョン内に生成します
    - ヒットポイントが継続されて状態異常は掛かっていない状態で生成されます
    - ダンジョンに空のセルが無い場合は生成されません
- `敵のみ付与される特殊能力`です

### 召喚
- ターン経過時に敵を生成します
    - 現在の階層に出現する敵が生成されます
    - この特殊能力は`1ターンに1度しか発動しません`
    - ダンジョンに空のセルが無い場合は生成されません
- `敵のみ付与される特殊能力`です

### 転生
- 死亡した際に他の敵を生成します
    - 現在の階層に出現する敵が生成されます
    - ダンジョンに空のセルが無い場合は生成されません
- `敵のみ付与される特殊能力`です

### 挑発
- 他の敵が受けるダメージを肩代わりします

### 血清
- デバフの毒を受けなくなります

### 集中
- デバフの呆けを受けなくなります

### 快癒
- デバフの負傷を受けなくなります

### 鍍金
- デバフの鈍らを受けなくなります

### 勇気
- デバフの恐怖を受けなくなります

### 魔除
- デバフの封印を受けなくなります

### 気付
- デバフの混乱を受けなくなります

### 再生付与
- 自分自身が出現した際に他の敵にバフの再生を付与します
- `敵のみ付与される特殊能力`です

### 鋭利付与
- 自分自身が出現した際に他の敵にバフの鋭利を付与します
- `敵のみ付与される特殊能力`です

### 硬化付与
- 自分自身が出現した際に他の敵にバフの硬化を付与します
- `敵のみ付与される特殊能力`です
